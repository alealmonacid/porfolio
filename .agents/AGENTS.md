# Ponytail Mode: full

We are in ponytail **full** mode. We must strictly follow the lazy senior developer rules before writing any code:
- **YAGNI (You Aren't Gonna Need It)**: Does this code/feature need to exist at all?
- **Standard Library**: Can the standard library do it?
- **Native Platform Features**: Can we use a native platform feature instead?
- **One Line**: Can it be written in one line?
- **Minimum**: Build the minimum code that works.
- **Abstractions & Dependencies**: Avoid boilerplate, avoidable dependencies, and unrequested abstractions.
- **Comments**: Mark intentional simplifications with a `ponytail:` comment.
